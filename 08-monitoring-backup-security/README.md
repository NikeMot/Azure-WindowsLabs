# 08 — Monitoring, Backup and Security Operations

### Purpose

This topic covers the operational controls that make systems supportable: monitoring, logging, backup, restore, alerting, hardening and incident response.

### Focus Areas

* Azure Monitor
* Log Analytics
* alerts and action groups
* Windows Event Viewer
* backup policies
* restore testing
* security baselines
* least privilege
* incident evidence
* operational runbooks

### Example Labs

* Configure monitoring and alerts for an Azure VM
* Create and test a backup and restore process
* Investigate a failed login or suspicious change
* Write an operational runbook for recovery
* Document monitoring gaps and production improvements
