# 03 — PowerShell Administration

### Purpose

This topic develops PowerShell as the main automation, reporting and evidence-gathering tool for Windows and Azure administration.

### Focus Areas

* PowerShell objects and pipeline
* discovery commands
* scripting structure
* functions and parameters
* error handling
* CSV and JSON processing
* scheduled tasks
* Windows administration automation
* Azure and Microsoft Graph PowerShell where appropriate

### Example Labs

* Build a server health-check script
* Generate AD user and group reports
* Automate starters, movers and leavers
* Query Event Viewer with PowerShell
* Create operational scripts with logging and error handling
