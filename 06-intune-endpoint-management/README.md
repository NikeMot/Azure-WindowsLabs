# 06 — Intune Endpoint Management

### Purpose

This topic covers modern endpoint management with Microsoft Intune, especially Windows device compliance, configuration, application deployment and troubleshooting.

### Focus Areas

* device enrolment
* compliance policies
* configuration profiles
* application deployment
* device actions and sync
* endpoint troubleshooting
* Conditional Access interaction
* modern workplace operations

### Example Labs

* Enrol a Windows device into Intune
* Configure compliance and investigate non-compliance
* Deploy an application to a test device group
* Troubleshoot a blocked user caused by device state
* Document endpoint support evidence
