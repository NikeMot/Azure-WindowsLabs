# 01 — Windows Server Foundations

### Purpose

This topic covers the base Windows Server administration skills needed before Active Directory, Azure hybrid work or production operations.

### Focus Areas

* Windows Server installation and initial configuration
* server roles and features
* services and startup behaviour
* patching and update basics
* local users and groups
* Windows Admin Center or Server Manager where useful
* Event Viewer and basic troubleshooting
* secure baseline thinking

### Example Labs

* Build a Windows Server baseline
* Configure server roles and document decisions
* Troubleshoot a failed Windows service
* Investigate boot, patching or event log issues

### Reference Material

* Windows Server 2022 and PowerShell
* The Practice of Cloud System Administration
* Operating Systems theory for services, processes, filesystems and resource management
* Microsoft Learn Windows Server documentation
