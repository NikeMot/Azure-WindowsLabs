# 05 — Entra ID Identity

### Purpose

This topic covers cloud identity administration using Microsoft Entra ID and connects traditional Windows identity knowledge to modern Microsoft cloud administration.

### Focus Areas

* users and groups
* administrative roles
* MFA and SSPR
* sign-in logs
* Conditional Access basics
* identity troubleshooting
* least privilege
* cloud identity versus AD DS

### Example Labs

* Create users and groups with role separation
* Configure MFA and SSPR for a test user population
* Investigate a failed sign-in
* Explain AD DS versus Entra ID for a support scenario
* Design least-privilege admin role assignments
