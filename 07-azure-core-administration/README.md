# 07 — Azure Core Administration

### Purpose

This topic covers the core Azure administration skills needed for cloud support, infrastructure operations and Azure administrator roles.

### Focus Areas

* resource groups
* RBAC
* Azure Policy basics
* VNets and subnets
* NSGs and secure access
* Azure VMs
* Azure Storage
* backup and cost controls
* Azure CLI and PowerShell verification

### Example Labs

* Deploy a secure Azure VM into a controlled VNet
* Configure RBAC and validate least privilege
* Troubleshoot VM access using NSGs and Network Watcher
* Configure Azure Storage securely
* Apply cost controls and cleanup procedures

### Reference Material

* Learning Microsoft Azure
* Azure Cookbook
* The Practice of Cloud System Administration
* Operating Systems theory for virtualisation, storage and networking abstraction
* Microsoft Learn Azure Administrator documentation
