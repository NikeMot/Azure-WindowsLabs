# 02 — Active Directory, DNS and Group Policy

### Purpose

This topic covers traditional enterprise identity and policy administration using AD DS, DNS and Group Policy.

### Focus Areas

* domain controllers
* AD DS domains and forests
* OUs, users, groups and computer objects
* group nesting and access models
* DNS zones and records
* Group Policy design and troubleshooting
* authentication and account lockout investigation
* domain-joined Windows clients

### Example Labs

* Build a domain controller and join a client
* Design OUs and groups for departments
* Configure and troubleshoot Group Policy
* Investigate DNS-related domain login failures
* Handle starters, movers and leavers with evidence

### Reference Material

* Active Directory Administration Cookbook
* Windows Server 2022 and PowerShell
* The Practice of Cloud System Administration
* Operating Systems theory for authentication, access control and name resolution dependencies
* Microsoft Learn Active Directory documentation
