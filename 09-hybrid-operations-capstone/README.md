# 09 — Hybrid Operations Capstone

### Purpose

This topic combines Windows Server, Active Directory, PowerShell, Entra ID, Intune, Azure, monitoring, backup and incident response into full operational scenarios.

### Focus Areas

* end-to-end Microsoft infrastructure administration
* hybrid identity reasoning
* operational troubleshooting
* change management
* incident response
* documentation and runbooks
* verification evidence
* production improvement planning

### Example Labs

* Build and support a small hybrid Microsoft environment
* Diagnose a user access issue across AD DS, Entra ID, Intune and Azure
* Recover from a simulated infrastructure failure
* Create a portfolio-ready final operations report
* Complete a timed practical assessment
