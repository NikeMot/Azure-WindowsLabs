# 04 — File Services and Permissions

### Purpose

This topic covers file services, NTFS permissions, shares and access troubleshooting, which are common Windows system administration responsibilities.

### Focus Areas

* file shares
* NTFS permissions
* share permissions
* inheritance
* group-based access
* access denied troubleshooting
* audit evidence
* backup and restore considerations

### Example Labs

* Build departmental file shares
* Configure least-privilege access using groups
* Troubleshoot a user access denied incident
* Compare share permissions and NTFS permissions
* Document a file access change request
